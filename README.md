# Hazelcast Packaging

Automation scripts to package and publish Hazelcast and Hazelcast Enterprise as DEB, RPM and Homebrew packages.

Installation instructions are available for [Community](https://docs.hazelcast.com/hazelcast/latest/getting-started/install-hazelcast#use-a-package-manager) and [Enterprise](https://docs.hazelcast.com/hazelcast/latest/getting-started/install-enterprise#use-enterprise-edition-binary) editions.
